Rezerwacja mieszkania

Projekt systemu wynajmu mieszkań ma na celu stworzenie aplikacji internetowej, która
umożliwi użytkownikom szybkie i bezpieczne wynajmowanie mieszkań. System powinien
być intuicyjny . Użytkownicy mogą przeglądać dostępne mieszkania, rezerwować terminy. 

 Proces realizacji wynajmu(Scenariusz)
 
1. Loguje się lub kontynuuje bez rejestracji.
2. Wybiera mieszkanie i termin wynajmu.
3. Użytkownik przegląda dostępne mieszkania.
4. Dokonuje rezerwacji.
5. Przechodzi do płatności.
6. Po sukcesie użytkownik otrzymuje potwierdzenie rezerwacji.
